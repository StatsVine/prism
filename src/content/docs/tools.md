The [prism-tools](https://github.com/StatsVine/prism-tools) repo contains the scripts that power the PRISM data pipeline. It is shared across all sibling repos and runs primarily in CI, but can be run locally against any checkout.

Tooling covers CSV validation, crosswalk and registry export builds, ID generation, and daily reconciliation checks against MLB rosters, the Chadwick Bureau, FanGraphs, and SFBB.

See the [README](https://github.com/StatsVine/prism-tools#readme) for setup and usage.
